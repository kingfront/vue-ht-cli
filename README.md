## vue-ht-cli
#框架介绍
基于vue-cli的快速规范开发框架,已封装常用组件,可直接进行基本项目开发

1,遵循eslint规则,提升代码质量

2,集成route使用示例

3,集成vuex可直接使用开发

4,封装axios、jsonp,util解决基本服务端接口请求

5,集成vant 移动端ui
